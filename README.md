# Redrob Ranker - team_solo

This repository contains the submission for the Redrob Ranker Hackathon.

## Setup Instructions

1. Ensure Python 3.10+ is installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Ensure the input data (`candidates.jsonl`) is located in `data/candidates.jsonl`.
4. Ensure the precomputed cache directory (`precompute/cache/`) contains the necessary cache files (these are generated by running the precompute scripts in order).

## Reproduction

To generate the `team_solo.csv` final ranking file, run the following single command from the repository root:

```bash
python rank.py
```

This command runs strictly offline on the CPU with zero network calls and executes in under 10 seconds. It produces a fully valid output file even if the LLM judgments cache is missing.
